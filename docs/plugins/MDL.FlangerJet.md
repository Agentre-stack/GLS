**Category:** Mod/Delay/LFO  
**Role:** Deep jet flanger with variable delay base, depth, and feedback.

## Signal Flow

`Input -> LFO-mod delay lines -> Feedback -> Tone mix -> Output`

## Parameters
- `delay_base`: Base delay (ms) for the comb notch.
- `depth`: Modulation amount.
- `rate`: LFO speed.
- `feedback`: Regeneration for jet sweeps.
- `manual`: Manual offset of the LFO.
- `mix`: Wet/dry blend.

## Usage Notes
- Increase feedback + depth for aggressive jet sweeps.
- Manual offset useful for tuning comb notch with static sweeps.
