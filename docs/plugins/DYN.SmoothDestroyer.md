# DYN.SmoothDestroyer — Phase 3 Cockpit

Goodluck cockpit for the dual-band dynamics/distortion shaper with trims, bypass, and presets.

## Controls
- **B1/B2 Freq / Q / Thresh / Range** — per-band centers, Q, thresholds, and gain range.
- **Attack / Release** — global envelope timing.
- **Mix** — blend processed signal with dry.
- **Input / Output Trim** — pre/post gain staging.
- **Soft Bypass** — UI bypass without releasing state.

## Factory Presets
- **Bus Tamer** — broad bus smoothing with moderate ranges.
- **Vocal De-Harsh** — upper-mid focus with tighter Q and deeper range.
- **Guitar Smooth** — low-mid and presence control with subtle trims.
