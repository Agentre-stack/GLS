# Chorus IX — MDL.ChorusIX

**Category:** Mod/Delay/LFO  
**Role:** Multi-voice chorus with spread, tone, and mix controls.

## Signal Flow

`Input -> Multi-voice delay (LFO mod) -> Tone filter -> Width mix -> Output`

## Parameters

- `voices`: 1–8 voice count.
- `rate`: LFO speed.
- `depth`: Modulation depth.
- `spread`: Stereo pan spread.
- `tone`: Low-pass tone tilt.
- `mix`: Wet/dry blend.

## Notes
- Depth and spread combine to widen backing vocals.
- Tone curve keeps shimmer controlled; set to negative on guitars.
