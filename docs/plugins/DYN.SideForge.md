# DYN.SideForge — Phase 3 Cockpit

Goodluck cockpit for the sidechain bus compressor with trims, bypass, and presets.

## Controls
- **Threshold / Ratio / Attack / Release** — main gain reduction shape.
- **SC HPF / SC LPF** — sidechain filter band.
- **Lookahead** — pre-delay for smoother gain moves.
- **Mix** — dry/wet blend.
- **Input / Output Trim** — pre/post gain staging.
- **Soft Bypass** — UI bypass without releasing state.

## Factory Presets
- **Bus Glue** — gentle bus compression with broad SC filters.
- **Drum Side** — faster clamp with tighter SC LPF and higher ratio.
- **Vocal Tight** — moderate ratio with upper-mid SC focus and light input trim.
