# DYN.VocalPin — Phase 3 Cockpit

Goodluck cockpit for the vocal leveler/de-esser with trims, presets, and soft bypass.

## Controls
- **Threshold / Ratio / Attack / Release** — main compressor controls.
- **DeEss Freq / Amount** — sibilance band and attenuation amount.
- **Mix** — dry/wet blend.
- **Input / Output Trim** — pre/post gain staging.
- **Soft Bypass** — UI bypass without releasing state.

## Factory Presets
- **Vocal Level** — medium ratio, balanced de-ess, slight makeup.
- **Air Tame** — higher de-ess focus with moderate compression.
- **Broadcast Pin** — lighter ratio with faster attack and small input trim.
