# DYN.TransFix — Phase 3 Cockpit

Goodluck cockpit for the transient tilt shaper with trims, presets, and soft bypass.

## Controls
- **Attack / Sustain** — boost/cut transient and body portions.
- **Tilt Freq / Tilt Amount** — spectral tilt pivot and amount.
- **Detect Mode** — Wideband, HF Focus, or LF Focus detection.
- **Mix** — dry/wet blend.
- **Input / Output Trim** — pre/post gain staging.
- **Soft Bypass** — UI bypass without releasing state.

## Factory Presets
- **Drum Snap** — HF-focused attack lift, lighter sustain.
- **Vocal Pop** — faster attack, mild tilt for vocal presence.
- **Bus Smooth** — softer attack with slight low tilt for buses.
