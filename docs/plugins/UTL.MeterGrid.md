# UTL.MeterGrid — Phase 3 Presets

Goodluck cockpit meter panel with K-scale presets and recallable meter behavior.

## Controls
- **Integration** — RMS window in ms.
- **Peak Hold** — peak hold duration in ms.
- **Scale** — Full Scale, K-12, K-14, or K-20 display.
- **Input / Output Trim** — pre/post gain.
- **Freeze** — stop meter decay.
- **Soft Bypass** — UI bypass without releasing state.

## Factory Presets
- **K-20 Broad** — 300 ms RMS, long peak hold, K-20 style ceiling.
- **K-14 Modern** — 400 ms RMS, moderate peak hold, K-14 display.
- **Broadcast** — 600 ms RMS, long peak hold, slightly lowered ceiling for broadcast checks.
