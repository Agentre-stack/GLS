# UTL.BandRouter — Phase 3 Presets

Goodluck cockpit for the three-way band router: split low/mid/high, pan/level per band, trim/mix, solos, and soft bypass with visual band meters.

## Controls
- **Low/Mid/High Level** — gain per band.
- **Low/High Split** — crossover points in Hz.
- **Low/Mid/High Pan** — pan each band within the stereo field.
- **Mix** — dry/wet blend.
- **Input / Output Trim** — pre/post gain staging.
- **Solo Low/Mid/High** — audition individual bands.
- **Soft Bypass** — UI bypass that leaves latency intact.

## Factory Presets
- **Mix Split** — neutral levels/pans with 120/3200 Hz splits.
- **Wide Mid** — subtle mid widening with tightened lows/highs.
- **Low Anchor** — boosted lows with anchored center, reduced highs.
